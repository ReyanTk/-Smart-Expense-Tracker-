# -Smart-Expense-Tracker-
Smart Expense Tracker

A Python-based smart expense tracking application that helps users manage daily spending, analyze financial habits, generate visual reports, and export professional PDF summaries.

Features
Add and manage expenses
Categorize transactions
Monthly spending summaries
Expense visualization with charts
PDF report generation
CSV-based data storage
Clean terminal interface
Modular and beginner-friendly code structure
Technologies Used
Python
pandas
matplotlib
reportlab
numpy
